# Limitations

## Major risks
- Construct mismatch between lab tasks and ecological behavior.
- Sparse direct evidence for many taxa, especially invertebrates.
- Taxonomy drift (name changes, splits, merges).
- Dataset and publication bias toward model/charismatic organisms.
- AI extraction can encode source bias if not strongly constrained.

## Mitigations
- Keep observed vs inferred values explicit.
- Report uncertainty bands for all estimated priors.
- Version taxonomy snapshots and synonym mappings.
- Track extraction confidence and require manual review thresholds.
